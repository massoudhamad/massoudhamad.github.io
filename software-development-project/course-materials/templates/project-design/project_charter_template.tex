\section{Success Criteria}
%============================================================

\begin{fillbox}[Definition of Done]
A user story is considered ``Done'' when:
\begin{itemize}
    \item[$\square$] Code is written and compiles without errors
    \item[$\square$] Feature is tested (manual or automated)
    \item[$\square$] Code is reviewed by at least one team member
    \item[$\square$] Code is pushed to main branch
    \item[$\square$] Feature is deployed to production
    \item[$\square$] Documentation is updated if needed
\end{itemize}
\end{fillbox}
